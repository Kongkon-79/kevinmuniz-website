 17. PROJECT STRUCTURE
================================================================================

  src/
  ├── app/
  │   ├── (auth)/                  # Auth pages — login, register, forgot-password
  │   ├── (website)/               # Public pages — home, campaigns, about, contact
  │   ├── api/auth/[...nextauth]/  # NextAuth API route
  │   ├── dashboard/
  │   │   ├── (backer)/            # Backer dashboard
  │   │   ├── (creator)/           # Creator dashboard
  │   │   └── (shared)/            # Shared dashboard — overview, settings, notifications

  │   ├── ui/                      # shadcn/ui components
  │   ├── shared/                  # Navbar, Footer, etc.
  │   ├── dashboard/               # Sidebar
  │   ├── modals/                  # Modal components
  │   ├── providers/               # AuthProvider, AppProvider
  │   └── utils/                   # Utility components
  ├── lib/                         # utils.ts (cn helper, etc.)
  └── middleware.ts                # Auth route protection


================================================================================
 18. QUICK SETUP COMMANDS (to reproduce from scratch)
================================================================================

  # 1. Create Next.js project
  npx create-next-app@14 ./ --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"

  # 2. Install shadcn/ui
  npx shadcn@latest init

  # 3. Add shadcn components
  npx shadcn@latest add accordion badge button calendar card carousel chart checkbox dialog dropdown-menu form input label pagination popover radio-group scroll-area select sheet skeleton sonner table textarea

  # 4. Install all dependencies
  npm install next-auth@^4 @tanstack/react-query @tanstack/react-table zustand axios react-hook-form @hookform/resolvers zod @tiptap/react @tiptap/starter-kit lucide-react motion date-fns react-day-picker sonner tailwind-merge tailwindcss-animate class-variance-authority clsx

  # 5. Create src structure
  mkdir -p src/app/{(auth),(website),api/auth,dashboard/{(backer),(creator),(shared)}}
  mkdir -p src/components/{ui,shared,dashboard,modals,providers,utils}
  mkdir -p src/lib
